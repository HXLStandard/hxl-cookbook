Reordering columns

View recipe in the HXL Proxy.

  1. We want the columns in the dataset to appear in a different order.

    Province Cluster District Organisation
    #adm1 #sector #adm2 #org
    Coast Province WASH Port District Org A
    Plains Province Education Grasslands District Org B
    Mountains Province Health Valley District Org A

  2. Create a separate, empty dataset—just the header and hashtag rows—with the columns in the order you'd like (leave out any columns you don't want included in the output):

    Province District Organisation Cluster
    #adm1 #sector #adm2 #org

  3. Choose the empty dataset as your primary dataset to process.

  4. Use the Append datasets filter to append your original dataset to the empty one (you may append multiple datasets if you'd like).
